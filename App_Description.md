### Complete Health & BMI Tracker Built with Flutter

I am excited to share my recent project: a comprehensive Health & BMI Calculator app. My goal was to build more than just a simple calculator—I wanted to create a fully functional, user-friendly health dashboard that helps users track their fitness progress over time while discovering actionable health insights.

**✨ Key Features:**
*   **Secure Authentication:** Integrated Firebase Authentication for seamless and secure user signup, login, and personalized experiences.
*   **Personalized Dashboard:** A dynamic main screen showing real-time BMI calculations, health status indicators, and quick access to daily data updates.
*   **Smart History Tracking:** Designed a robust local database that safely logs and isolates each user's historical weight and BMI data across different sessions, complete with time-filtering and intuitive empty-state handling.
*   **Comprehensive Profile Management:** Users can customize their avatar and details, set fitness goals (Target Weight & Target BMI), and easily toggle between Metric (kg/cm) and Imperial (lbs/ft) measurement units on the fly.
*   **Health Insights Library:** A dedicated, fully searchable section loaded with categorized health modules—ranging from nutrition basics to cardio health—designed to encourage a healthier lifestyle.

**🛠️ Tech Stack & Tools:**
*   **Framework:** Flutter & Dart for building a highly responsive, cross-platform UI with a native feel.
*   **State Management:** GetX for efficient, reactive state management, snappy routing, and dependency injection.
*   **Backend & Security:** Firebase Authentication for modern, secure user management.
*   **Local Storage:** SQLite for handling relational history data (isolated by User ID), paired with GetStorage for rapid access to user preferences and settings.
*   **UI/UX Enhancements:** Implemented modern design principles, fluid micro-animations, and interactive components for a premium, professional user experience.

This project was a fantastic opportunity to strengthen my skills in complex state management, local-to-cloud data orchestration, and building production-ready user interfaces. It is designed to be clean, intuitive, and highly scalable!
